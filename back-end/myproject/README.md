# 정하의 첫 개인 프로젝트
개인의 역량을 키우기 위한 프로젝트입니다.
  - 첫번째 목표: `Hello world!!!`
  - 두번째 목표: `회원가입 구현하기`
  - 세번째 목표: `회원가입 상세구현하기`
    :react-router 이용해서 회원가입에 성공한 user를 로그인 화면으로 이동시키기
